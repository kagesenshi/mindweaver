## What is Mindweaver?

Mindweaver provides a metadata layer on top of Kubernetes and ArgoCD to manage core data warehousing, data lake, and AI platform components. It allows you to:

- Orchestrate the deployment of complex data services.
- Manage projects and their associated resources.
- Centralize metadata for data sources and AI agents.
- Handle sensitive information securely.
